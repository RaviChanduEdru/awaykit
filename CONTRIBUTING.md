# Contributing to awaykit

Thanks for your interest! awaykit is early — the best contributions right now:

1. **Design review** — poke holes in docs/SECURITY.md and docs/ARCHITECTURE.md
2. **Agent adapters** — knowledge of Codex / Cursor CLI / OpenCode hook systems
3. **Mobile** — React Native expertise for the approval-card UX

## Workflow
- Open an issue before large PRs
- Conventional commits (`feat:`, `fix:`, `docs:`)
- Security issues → private GitHub security advisory, not public issues

## Dev setup
```bash
npm install
npm run dev:daemon
```
